# Touche pas au klaxion

Application de covoiturage interne réalisée en PHP MVC.

## Fonctionnalités
- Affichage des trajets
- Réservation d’un trajet
- Création de trajets
- Suppression de trajets
- Gestion administrateur

## Technologies
- PHP
- MySQL
- Bootstrap
- MVC

## Installation
1. Cloner le projet
2. Importer la base de données
3. Lancer XAMPP
4. Ouvrir localhost/touche_pas_au_klaxion/public
