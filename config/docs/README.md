# Touche pas au klaxon

Application PHP/MySQL de gestion de covoiturage en entreprise.

## Fonctionnalités

- Connexion utilisateur/admin
- Création de trajets
- Réservation de trajets
- Gestion des agences
- Gestion des utilisateurs
- Tableau d’administration
- Modification et suppression

## Technologies

- PHP
- MySQL
- Bootstrap
- HTML/CSS

## Installation

1. Copier le projet dans htdocs
2. Importer les fichiers SQL
3. Lancer Apache/MySQL
4. Ouvrir :

http://localhost/touche_pas_au_klaxon/public/
